# Skybow

### Purpose
The skybow is the common bow wielded by elven archers.

### Physical Description
A skybow has a frame made out of white oak wood. The string is a single thread of wind silk, which is one of the strongest materials. This special silk allows for much farther range and more precision than a conventional bow. On one side of the bow shaft, a sharp spear point is mounted for defense of the wielder should an enemy get close.
